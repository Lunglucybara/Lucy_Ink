# 루시잉크 v0.4-diary · 다이어리 모드 미감 전면 개선

다이어리 모드 홈과 설정/편집 화면의 **디자인 언어를 새로 구축**했습니다.
브랜드 컨셉(Lucid Drop · 젬 물방울 + 펜촉)에서 출발한 **"잉크 & 젬 스테이셔너리"** 방향으로,
평면적이던 기존 UI에 깊이·질감·정제된 타이포그래피를 입혔습니다. 라이트/다크 모두 개별 튜닝했습니다.

## 바뀐 것

### 히어로(다이어리 커버)
- 사선 젬 컷 광택 + 우상단 액센트 광휘(radial glow)로 "커버" 질감 부여
- 하단에 액센트 심(seam) 라인, 상단 엠보싱 하이라이트 추가
- `DIARY` 아이브로우를 Parisienne 스크립트로 교체(브랜드 워드와 통일)
- 썸네일에 젬 페이싯 글린트, 툴 버튼에 호버 리프트 인터랙션

### 위젯 카드
- 카드마다 엠보싱 상단 하이라이트 + 다층 그림자로 실제로 눌린 듯한 입체감
- 헤더 아래 오른쪽으로 페이드되는 얇은 액센트 심
- 위젯 아이콘 칩을 젬 컷 그라디언트 + 내부 하이라이트로 리터치
- `+` 추가 버튼 호버 시 90° 회전 마이크로 인터랙션

### D-데이 카드
- 숫자 타이포 강화(태뷸러 넘버, 굵기·자간 조정), 코너 페이싯 글린트
- 3종 템플릿(이미지/글래스/페이퍼)이 서로 명확히 구분되도록 재설계
  - 글래스: 실제 프로스트 느낌, 페이퍼: 종이 그레인 오버레이
- 호버 리프트

### 달력
- 오늘 표시를 채워진 젬 도트(라디얼 + 글로우)로 변경
- 일정/할일 마커 도트에 은은한 발광, 일요일 헤더 색 구분
- 셀 호버 상태 추가

### 리스트·투두·주간·뽀모도로
- 스케줄/목록 행에 리딩 액센트 틱 + 호버 슬라이드
- 투두 완료 항목은 액센트 톤 취소선, 기간 배지를 알약형으로 정리
- 주간 "오늘" 날짜 칩 강조
- 뽀모도로 링에 내부 액센트 링 + 광택, 시계 타이포 확대

## 보존 원칙 준수
- 클래스명·id·DOM 구조·데이터 속성 훅 **일절 변경 없음** — 시각 선언 강화 및 장식 규칙 추가만 수행
- 정의만 되어 있고 값이 비어 있던 `--accent-ink`를 다이어리 스코프에 안전하게 채워 대비 개선(다른 화면 영향 없음)
- 외부 CSS 로드 순서 충돌 없음(다이어리 스타일은 전부 인라인)

## 검증
- Playwright headless Chromium, 뷰포트 412×915 / dpr2

- 스타일 블록 중괄호 균형 및 다이어리 선택자 유실 여부 점검 통과
