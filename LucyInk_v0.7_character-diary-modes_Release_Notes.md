# Lucy Ink v0.7 — 캐릭터·모드 분리 업데이트

## 메모 / 다이어리 모드
- 메모 모드와 다이어리 모드의 퀵메뉴 설정을 완전히 분리했습니다.
- 메모 모드 퀵메뉴에는 **다이어리 모드 전환**을, 다이어리 모드 퀵메뉴에는 디데이·투두·일정·다이어리 관리용 기능을 넣을 수 있습니다.
- 설정을 **공통 / 메모 모드 / 다이어리 모드** 탭으로 나누었습니다.

## 캐릭터 카드
- 새 **캐릭터 / 다인 캐릭터**는 이미지형과 텍스트형을 선택할 수 있습니다.
- 이미지형은 기존 포트레이트·정사각 썸네일 흐름을 사용합니다.
- 텍스트형은 포트레이트·정사각 썸네일을 숨기고, 제목이 있는 설명 항목을 여러 개 구성할 수 있습니다. 갤러리는 유지하며 보기 화면에서 이미지가 없으면 숨깁니다.
- 기존 캐릭터 카드는 데이터 호환성을 유지한 채 **루미 캐릭터 / 루미 다인 캐릭터**로 표시됩니다.
- 캐릭터 상세 설명의 토큰 표기는 공백 포함·공백 제외 글자 수 표기로 변경했습니다.

## 저장 호환
- 전체 백업과 자동 백업에 메모 퀵메뉴와 다이어리 퀵메뉴를 각각 기록합니다.
- 이전 백업은 다이어리 퀵메뉴가 없는 상태로도 정상 복원됩니다.
